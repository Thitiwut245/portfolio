Nurse Learning Platform

An educational simulation for nursing students.

Project Overview

A serious-game simulation for training nursing students in clinical protocols. This project validates student actions against strict medical procedures, providing immediate feedback based on clinical decision-making.

Details & Responsibilities

Core Interaction System

Implemented a clean and responsive interaction system for manipulating medical tools and patient avatars.

Scenario Management

Architected a Finite State Machine (FSM) to track procedure progress, validating steps against medical protocols and handling branching failure states.

Bone Animate and Asset Creation

Implemented 2D bone animation controllers to visualize complex medical procedures dynamically.
