Tech Demo 2D

An ongoing 2D technical demonstration.

Project Overview

Tech Demo 2D is an ongoing 2D gameplay systems project developed in Unity. This project was created after transitioning from GDScript and Godot, with the primary goal of building practical understanding of Unity’s gameplay systems and C#-based development workflow. The project functions as a technical sandbox for implementing and iterating on core gameplay systems such as player movement, combat logic, enemy behavior, and UI interaction. Emphasis is placed on clean architecture, modular system design, and iterative improvement rather than narrative or content completeness. New systems are added, tested, and refined continuously as the project evolves.

Details & Responsibilities

Gameplay Systems

Implemented core 2D gameplay mechanics, including player movement, jumping, aiming, and responsive input handling.

Core Mechanics
  • Physics-Based Movement: Smooth acceleration and deceleration for character movement (PlayerMovement2D.cs).
  • Advanced Jumping: Features coyote time, jump buffering, and variable jump height for responsive platforming.
  • Aiming & Combat: Support for mouse aiming and directional facing, allowing for precise weapon control (Shotgun.cs).
  • Input System: Decoupled input handling using PlayerInputReader.cs to manage control schemes efficiently.
Implementation Details

Horizontal Movement

Smooth acceleration and deceleration using custom friction logic:

private void HandleHorizontalMovement() { float targetSpeed = input.HorizontalInput * moveSpeed; float currentSpeed = rb.linearVelocity.x; float speedDifference = targetSpeed - currentSpeed; float accelRate = decel; if (Mathf.Abs(targetSpeed) > 0.01f) { accelRate = accel; } float movement = speedDifference * accelRate * Time.fixedDeltaTime; rb.linearVelocity = new Vector2(currentSpeed + movement, rb.linearVelocity.y); }

Responsive Jumping

Uses a jump buffer and coyote time for better game feel:

private void JumpCheck() { // check if we can jump if (jumpBufferCounter > 0f) { if (coyoteCounter > 0f) { Debug.Log("Jump!"); rb.linearVelocity = new Vector2(rb.linearVelocity.x, jumpForce); jumpBufferCounter = 0f; coyoteCounter = 0f; } } }

Variable Jump Height & Gravity

Applies different gravity multipliers for falling and "low jumps" (releasing space early):

private void ApplyGravity() { if (rb.linearVelocity.y < 0f) { // Falling rb.linearVelocity += Vector2.up * Physics2D.gravity.y * (fallMultiplier - 1f) * Time.fixedDeltaTime; } else if (rb.linearVelocity.y > 0f && !input.JumpHeld) { // Rising but jump not held rb.linearVelocity += Vector2.up * Physics2D.gravity.y * (lowJumpMultiplier - 1f) * Time.fixedDeltaTime; } }
